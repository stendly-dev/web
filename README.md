# Stendly Web App

> READ-ONLY MIRROR: this repository is automatically synced from the private Stendly monorepo.
> Do not open pull requests here. Use the main Stendly repository or the community issue tracker.

Stendly web frontend for wallet flows, checkout and pay links, and merchant B2B tools for non-custodial USDC payments on Solana.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
